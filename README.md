# Replay Skills

A collection of skills for working with [Replay](https://replay.io) tools.

## Available Skills

- **replay-cli** - Record your application to gather performance data and debug issues using the Replay CLI
- **replay-mcp** - Use Replay MCP to inspect the contents of https://replay.io recordings.

## Documentation

For more information about Replay, visit [docs.replay.io](https://docs.replay.io)
